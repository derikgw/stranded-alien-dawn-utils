SeedGeneratorApp - User Documentation

Overview:

SeedGeneratorApp is a graphical user interface (GUI) application designed to generate seeds for PC games. The seeds are generated using words from the WordNet database stored in a local SQLite database. The application provides functionalities such as generating seeds, copying them to the clipboard, and creating/recreating the word database.

Prerequisites:

  • Python: Ensure you have Python installed on your machine.
  • Required Python Libraries:
    • tkinter - for GUI components.
    • pyperclip - to copy seed data to the clipboard.
    • PIL (from the Pillow package) - for image processing.
    • threading - for background tasks without freezing the GUI.
    • nltk - for accessing the WordNet database.
    • sqlite3 - for database operations.
    Install these libraries using pip: 
    pip install tkinter pyperclip Pillow nltk sqlite3
  • WordNet Data: Before using the application, ensure WordNet data is downloaded. The script contains a method (ensure_wordnet_data()) that checks and downloads the WordNet data if not already present.

How to Run:

  1. Navigate to the directory containing the generate_seed_gui.pyw script.
  2. Run the script: 
    python generate_seed_gui.pyw

Features:

  • Banner: Displays a banner image from the 'images/banner.png' path.
  • Create/Recreate Database Button:
    • Initializes or recreates the word database used for generating seeds. Displays progress in a progress bar.
  • Generate Seed Button:
    • Produces a new game seed using words from the database and displays it. The seed generation uses a separate script that fetches random words from the SQLite database and combines them.
  • Copy to Clipboard Button:
    • Allows users to copy the current seed to the system clipboard.
  • Progress Bar:
    • Shows database creation progress when the "Create/Recreate Database" button is clicked.
  • Word Count Display:
    • Displays the current count of words in the database.
  • Cancel Button:
    • Halts the ongoing database creation process.

Word Database:

The application utilizes a SQLite database to store words fetched from the WordNet database. The database is named word_database.db and contains a table words where individual words from WordNet are stored. The create_database() function checks if the database file exists, and if so, the words table is dropped and recreated. This ensures fresh and updated words for generating seeds.

Game Seed Generation:

The application uses a separate script for seed generation. This script fetches two random words from the SQLite database and combines them with a hyphen to form the game seed. The methods involved include:

  • fetch_random_word_from_database(): Retrieves a random word from the SQLite database.
  • generate_game_seed_from_database(): Combines two fetched words with a hyphen to produce the game seed.

Workflow:

  1. If running the application for the first time or refreshing the word database, click "Create/Recreate Database".
  2. Click "Generate Seed" to produce a new game seed.
  3. Use "Copy to Clipboard" to copy the generated seed for your game or other applications.
